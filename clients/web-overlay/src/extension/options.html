彈幕 Overlay 設定

隱私同意

彈幕 Overlay 會處理暱稱、彈幕訊息、隨機裝置識別碼、房間憑證及連線 IP，以提供聊天室、房間管理及防止濫用功能。擴充功能不會讀取或傳送目前網頁內容。

網站存取說明

為了在一般 HTTP/HTTPS 網頁顯示自己的 Shadow DOM Overlay，擴充套件需要廣泛網站權限。它不讀取或傳送頁面正文、表單、Cookie、localStorage 或網址參數，也不建立頁面與擴充套件間的任意訊息橋接。

隱私與安全

房主管理憑證只保存在背景 trusted storage，content script 與網頁都無法取得。撤回同意後會立即中斷連線並移除隨機裝置識別碼。
