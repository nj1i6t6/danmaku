# 可接受使用政策

版本：1.0｜生效日期：2026-07-15

本政策適用於專案維護者明確標示的官方服務。自架者應採用並依當地法律調整自己的政策；第三方服務不是官方服務，也不得暗示原作者為其內容或營運背書。

## 允許的使用

在遵守 [LICENSE](LICENSE)、[TERMS-OF-SERVICE.md](TERMS-OF-SERVICE.md) 與適用法律的前提下，可把服務用於非商用的即時討論、社群互動、教育、研究、個人或興趣用途。

## 禁止行為

不得使用本軟體或官方服務：

1. 從事違法、詐欺、侵權、威脅、騷擾、仇恨、跟蹤、剝削或鼓勵自傷／暴力的行為。
2. 發布兒童性剝削、未經同意的私密內容、人口販運、恐怖活動或其他依法禁止內容。
3. 冒充他人、官方專案、維護者或可信機構，或偽造背書、來源及安全狀態。
4. 發送垃圾訊息、重複內容、惡意連結、釣魚、惡意程式、聯絡方式導流、廣告或未經同意的招攬。
5. 收集、揭露或交易他人的個人資料、憑證、房間密碼、owner credential、token 或其他秘密。
6. 侵犯著作權、商標、隱私、公開權、營業秘密或其他第三人權利。
7. 規避內容檢查、typed ACK、房間容量、rate limit、檢舉限制、Origin/CSP、權限邊界或其他安全控制。
8. 進行阻斷服務、連線洪水、未經授權掃描、反向工程他人秘密、存取他人房主管理功能或干擾基礎設施。
9. 利用 Overlay 誘導使用者在宿主頁面輸入秘密、遮蔽重要安全提示，或偽裝成宿主網站／瀏覽器 UI。
10. 直接或間接商用。商用限制的完整定義與例外以 [LICENSE](LICENSE) 為準。

## 內容處理

服務可能依自動規則拒絕、降速、shadow-drop、暫時靜音、隱藏或排隊內容，也可能因多次檢舉隱藏訊息。自動規則無法保證攔截全部有害內容，也可能誤判。

維護者可為安全、法令、容量與社群保護而移除內容、限制連線、刪除房間或停止服務，但沒有持續監看所有內容的義務。緊急危險請聯絡當地緊急服務，不要依賴聊天室處理。

只檢舉你合理認為違反本政策的近期訊息。不得以檢舉騷擾他人、操縱三人隱藏機制或耗盡檢舉容量。安全漏洞請依 [SECURITY.md](SECURITY.md) 私下通報，不要在彈幕中測試或揭露。

## 執行與申訴

違反本政策可能導致內容隱藏、冷卻、斷線、房間刪除、來源封鎖、保留證據或依法通報。措施可依嚴重程度立即採取。

若官方網站或 repository profile 提供實際聯絡管道，可透過該管道提出申訴並附上時間、房間碼、message ID 與簡短理由；不要附 owner credential 或第三人敏感資料。沒有帳號的架構可能限制身分驗證與復原能力。
