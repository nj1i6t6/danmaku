—

顯示，而不是接管

Overlay 只佔一層視覺空間，不把你送進需要重新理解的聊天介面。

—

跨裝置，但不跨界

每個入口各自遵循平台的安裝與權限習慣，不以一套體驗硬套所有螢幕。

—

透明的預設

下載尚未發布時就清楚標示準備中；連結、權限與發布狀態都不藏在按鈕背後。

PRIVACY BY DESIGN

只做顯示所需的事

彈幕可以很熱鬧，權限說明不必模糊。

安全

權限有邊界

瀏覽器版本安裝時會要求一般 HTTP 與 HTTPS 網站權限，才能在已授權網站自動顯示 Overlay。

此權限只用來建立彈幕 Overlay，不讀取、上傳或分析宿主頁面內容、表單、Cookie 或瀏覽資料。

靜態入口

零後端請求

純前端頁面

此頁只載入同源靜態資產；不請求健康端點，也不建立即時連線。

READ BEFORE INSTALL

把必要的話，放在入口旁邊

正式版本會持續補齊對應文件；在此之前，先保留清楚可到達的導覽位置。
